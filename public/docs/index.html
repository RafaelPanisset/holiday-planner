Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Auth

Authentication API

Register a user

Registers a new user and returns an access token

Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"quod\",
    \"email\": \"frederique68@example.net\",
    \"password\": \"&x$;cpMhxm\",
    \"password_confirmation\": \"sapiente\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "quod",
    "email": "frederique68@example.net",
    "password": "&x$;cpMhxm",
    "password_confirmation": "sapiente"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "token": "access_token"
}

Example response (422):


{
    "error": "Problem in the validation!",
    "validation": {
        "name": [
            "The name field is required."
        ],
        "email": [
            "The email field is required.",
            "The email has already been taken."
        ],
        "password": [
            "The password field is required.",
            "The password must be at least 8 characters."
        ],
        "password_confirmation": [
            "The password confirmation does not match."
        ]
    }
}
Received response:
Request failed with error:


Tip: Check that you're properly connected to the network.
If you're a maintainer of ths API, verify that your API is running and you've enabled CORS.
You can check the Dev Tools console for debugging information.

Login a user

Logs in a user and returns an access token

Example request:

    "http://localhost/api/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"nikolas.kertzmann@example.org\",
    \"password\": \"NdA.U^BBL%5b\\\\~\'(^;>&\"
}"

const url = new URL(
    "http://localhost/api/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {

    \"description\": \"Cupiditate nihil enim quo dolorum earum aperiam veniam excepturi.\",
    \"date\": \"iste\",
    \"location\": \"ut\",
    \"participants\": [
        \"quasi\"
    ]
}"

const url = new URL(
    "http://localhost/api/holiday-plans"
);
